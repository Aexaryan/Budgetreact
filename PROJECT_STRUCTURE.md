## 🚀 **How to Run the Organized Project**

### 1. **Start the Server**
```bash
cd backend
npm install
npm start
```

### 2. **Access the Application**
- **Main Hub**: `http://localhost:3000`
- **Accounting System**: `http://localhost:3000/accounting`
- **Direct Pages**: `http://localhost:3000/frontend/pages/[page-name].html`
